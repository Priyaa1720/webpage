Learn JAVA in fun way!

Java is a high-level, object-oriented, class-based programming language developed by Sun Microsystems (now owned by Oracle). It was released in 1995 and is designed to be platform-independent — meaning code written in Java can run on any system that supports the Java Virtual Machine (JVM).
